1) pom.xml 에서 관련 라이브러리 의존성을 설정한다 (mybatis, mybatis-spring, spring-orm, hikariCP)

2) 요청 주소에 따라 실행할 컨트롤러의 함수를 구성한다

3) 컨트롤러 -> 서비스 -> DAO(인터페이스)
- 인터페이스 DAO가 정상적으로 작동하려면, root-context.xml 에서 
  <mybatis-spring:scan base-package="com.itbank.model" /> 형식으로 인터페이스 DAO가 있는 패키지를 지정해야 한다
- 인터페이스에 의해 구동시 마이바티스 프록시 객체가 생성되어 작동한다

4) 인터페이스 DAO는 어노테이션으로 쿼리문을 지정하거나, xml과 연동할 수 있다
@Select, @Insert, @Update, @Delete 어노테이션을 함수위에 작성해준다
어노테이션에 괄호를 열고 문자열 형식으로 원하는 SQL 구문을 작성하면 된다
select 는 반환형을 반드시 명시해야 한다
insert, update, delete는 반환형이 자동으로 int 형으로 고정된다

마이바티스 인터페이스 DAO에 파라미터를 전달할때는 가급적 하나만 전달한다
여러개의 파라미터를 전달하려면 HashMap으로 묶어서 전달한다
